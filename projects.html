Rustlets

Container Runtime in Rust

Rust Linux Namespaces cgroups

Built a lightweight container runtime in Rust that leverages 8 Linux namespaces to achieve process isolation with sub-100ms startup times, significantly faster than traditional virtual machines.

Key Features

  • Resource management through cgroups v2 integration, enabling precise CPU and memory limits with less than 5% overhead
  • Support for deploying 100+ concurrent containers per host
  • Security layers including capability dropping, seccomp-BPF syscall filtering, and pivot_root filesystem isolation
  • Reduced attack surface by 95% compared to unprivileged processes
